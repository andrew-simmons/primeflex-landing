Form Layout

Form layout is a CSS utility optimized for creating forms with ease.

Getting Started

Core member of the FormLayout is the .p-field class that wraps the input field and the associated label.

Vertical Layout

In its simplest form, a vertical layout is created when used within .fluid that makes the components use all available width in the following demo.

Vertical

<h5>Vertical</h5>
<div class="fluid">
    <div class="p-field">
        <label for="firstname1">Firstname</label>
        <input id="firstname1" class="inputfield" type="text"> 
    </div>
    <div class="p-field">
        <label for="lastname1">Lastname</label>
        <input id="lastname1" class="inputfield" type="text"> 
    </div>
</div>
Vertical Layout with Grid

This is where FormLayout actually hooks-in to PrimeFlex with the help of .p-formgrid class to optimize the content for form design. Example below arranges two fields to be displayed next two each other.

Vertical and Grid

<h5>Vertical and Grid</h5>
<div class="fluid p-formgrid p-grid">
        <div class="p-field p-col">
        <label for="firstname2">Firstname</label>
        <input id="firstname2" class="inputfield" type="text"> 
    </div>
    <div class="p-field p-col">
        <label for="lastname2">Lastname</label>
        <input id="lastname2" class="inputfield" type="text"> 
    </div>
</div>
Horizontal and Fixed Width

In horizontal form, label of the field is displayed on the same row of the input as opposed to the vertical alternative. In this example, label has a fixed width where container of the inputs gets the remaining space.

Horizontal and Fixed Width

<h5>Horizontal and Fixed Width</h5>
<div class="p-field p-grid">
    <label for="firstname3" class="p-col-fixed" style="width:100px">Firstname</label>
    <div class="p-col">
        <input id="firstname3" class="inputfield" type="text"> 
    </div>
</div>
<div class="p-field p-grid">
    <label for="lastname3" class="p-col-fixed" style="width:100px">Lastname</label>
    <div class="p-col">
        <input id="lastname3" class="inputfield" type="text"> 
    </div>
</div>
Horizontal and Fluid

Wrapping the form in the previous example with .fluid and removing the fixed width results in a fluid layout.

Horizontal and Fluid

<h5>Horizontal and Fluid</h5>
<div class="fluid">
    <div class="p-field p-grid">
        <label for="firstname4" class="p-col-12 p-mb-2 p-md-2 p-mb-md-0">Firstname</label>
        <div class="p-col-12 p-md-10">
            <input id="firstname4" class="inputfield" type="text"> 
        </div>
    </div>
    <div class="p-field p-grid">
        <label for="lastname4" class="p-col-12 p-mb-2 p-md-2 p-mb-md-0">Lastname</label>
        <div class="p-col-12 p-md-10">
            <input id="lastname4" class="inputfield" type="text"> 
        </div>
    </div>
</div>
Inline

Inline forms are used to display the content on the same row and can simply be implemented by adding .p-formgroup-inline to the form container. Note that per design requirements, if labels are not visually hidden, it is suggested to use .p-sr-only to still support screen readers.

Inline

<h5>Inline</h5>
<div class="p-formgroup-inline">
    <div class="p-field">
        <label for="firstname5">Firstname</label>
        <input id="firstname5" class="inputfield" type="text" placeholder="Firstname"> 
    </div>
    <div class="p-field">
        <label for="lastname5">Lastname</label>
        <input id="lastname5" class="inputfield" type="text" placeholder="Lastname"> 
    </div>
    <button class="button" type="button">Submit</button>
</div>
Checkbox and RadioButton

Checkbox and RadioButton have exclusive layout support via .p-field-checkbox and .p-field-radiobutton classes respectively. Examples here demonstrates vertical and horizontal layout alternatives.

Vertical Checkbox
Horizontal Checkbox

</div>
<div class="p-field-checkbox">
    <input type="checkbox" name="city2" id="city2"></input>

Helper text

Helper text is an optional element defined with the small tag to display additional information about the input field.

Help Text
Enter your username to reset your password. 

<h5>Help Text</h5>
<div class="p-field fluid">
    <label for="username">Username</label>
    <input id="username" class="inputfield" type="username" aria-describedby="username-help"> 
    <small id="username-help">Enter your username to reset your password.</small>
</div>
Advanced Forms

A responsive form with various input fields can easily be implemented using a combination of .p-field, .p-formgrid and .p-fluid.

Advanced

<h5>Advanced</h5>
<div class="fluid p-formgrid p-grid">
    <div class="p-field p-col-12 p-md-6">
        <label for="firstname6">Firstname</label>
        <input class="inputfield" id="firstname6" type="text"> 
    </div>
    <div class="p-field p-col-12 p-md-6">
        <label for="lastname6">Lastname</label>
        <input class="inputfield" id="lastname6" type="text"> 
    </div>
    <div class="p-field p-col-12">
        <label for="address">Address</label>
        <textarea class="inputfield" id="address" type="text" rows="4"></textarea>
    </div>
</div>
Customization

FormLayout comes with sensible defaults, in case these values need to be customized suggested way is building primeflex.scss with your on variables.

Variable Default
$fieldMargin 1em
$fieldLabelMargin .5em
